# Hiking web app
</div>

### About The Project

The project is a responsive web application designed to help users discover hikingtrails across Sweden through an interactive map interface. You can also login as admin to create new trails to the database.

### Built With

Frontend: Next.js (App Router) + TypeScript
Styling: Tailwind CSS
Maps: Leaflet
Backend: Supabase (PostgreSQL, Auth, API)
Deployment: Vercel
