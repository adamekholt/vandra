### About The Project

Responsive MVP web application for discovering hiking trails in Sweden through an interactive map interface. Users can explore nearby trails, filter by difficulty and length, search locations, and view trail details. Administrators can log in and manage trail data.

### Built With

Next.js (App Router), Supabase (auth/database/API), TypeScript, Tailwind CSS, Leaflet (interactive maps). Deploys to Vercel.
